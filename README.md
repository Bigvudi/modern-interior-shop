# 🪑 Интернет-магазин мебели

Адаптивный сайт интернет-магазина мебели. Верстка выполнена по макету, включает 3 страницы.

![Превью проекта](/img/preview.jpg)

[**Смотреть онлайн**](https://github.com/Bigvudi/modern-interior-shop.git)

## 📄 Страницы проекта

| Страница | Описание |
|----------|----------|
| Главная | Промо-блок, категории мебели, товары со скидками |
| Каталог | Сетка товаров, фильтры, сортировка |
| Корзина | Список товаров, изменение количества, итог |

## 🔧 Функционал

- Шапка: логотип, поиск, навигация
- Выпадающая корзина с товарами
- Карточки товаров: фото, название, описание, цена, скидка
- Иконки "Избранное" и "В корзину"
- Категории мебели (отдых, работа, кухня, детская, ванная)
- Подвал: контакты, соцсети, подписка

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- SCSS
- Адаптивная верстка
- Семантическая верстка

## ✨ Особенности

- Чистый код
- Корректное отображение на всех устройствах
- Hover-эффекты
- Выпадающие меню
